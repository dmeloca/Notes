Linux Basics
------------
- What is Linux?
Is an OS, that require little hardware resources and maintained by the community. Designed to write
and use network-based applications. Linux is Open Source which menas everyone could get the kernel's
source code to inspect it, modify it, recompiled and redistribute the program.

A Linux distribution include the Linux kernel with customized tools and software packages.
